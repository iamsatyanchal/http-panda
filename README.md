**Pandas for every HTTP response status code.**

http-pandas is a fun and simple project that shows a realistic panda image for each HTTP status code (like 200, 404, 500, etc).  
Each image visually represents the meaning of the status code in a meme-style format.

Visit at: https://http-pandas.vercel.app

You can directly access images using this format: https://http-pandas.vercel.app/[status_code].jpg

**Example:**

https://http-pandas.vercel.app/404.jpg

https://http-pandas.vercel.app/202.jpg
